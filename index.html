togg

Github npm
Angular UI directive that uses events to toggle classes, attributes, properties. It helps developers separate UI interaction from application code and therefore reduces prototyping and engineering time because UI components can be created separately from how their functionality is hooked up.

Interactive Examples

  • Click to toggle own class

  • Click to toggle classes on targets

  • Click to toggle classes in named scope

  • Click to enable/disable input

  • Check/uncheck to toggle attribute

    checked unchecked

  • Click to trigger custom event

  • Click to empty target
